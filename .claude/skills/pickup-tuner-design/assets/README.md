# Assets

Pickup Tuner is a code-drawn egui desktop app — the source repo
(`ncermesoni/pickup-tuner`) ships **no raster or vector image assets**: no logo
file, no photography, no illustrations. Everything visible is drawn live from
audio data (meters, tuner dial, heatmap grid).

Accordingly there are no binaries to copy here. The brand mark is expressed in
CSS as a monospace wordmark with a small bar-graph glyph — see
`guidelines/brand-wordmark.card.html`.

If a real logo or product imagery is later produced, drop the files in this
folder and update the **Iconography** / **Index** sections of `readme.md`.
